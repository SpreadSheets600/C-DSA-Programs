# Stacks And Queues

Stacks and queues are linear data structures.

They both store items in order, but they remove items in different ways.

## Fast Idea

| Structure | Rule |
|-----------|------|
| Stack | Last In, First Out (LIFO) |
| Queue | First In, First Out (FIFO) |
| Priority Queue | Higher-priority item gets served first |

## Real-Life Picture

- Stack: pile of plates
- Queue: line of people
- Circular Queue: chairs around a round table
- Priority Queue: emergency patients, where urgent cases go first
